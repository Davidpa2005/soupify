SOUPIFY
Aplicación web para la gestión de playlists musicales

Soupify es una aplicación web desarrollada como proyecto académico en la asignatura Desarrollo de Webs y Apps del Grado en Ingeniería Informática de la Universidad Europea de Madrid. El objetivo del proyecto es ofrecer una plataforma sencilla e intuitiva que permita a los usuarios descubrir música, reproducir canciones y organizar playlists personalizadas, aplicando los principios fundamentales del desarrollo web.

La aplicación ha sido desarrollada desde cero, sin el uso de frameworks, utilizando tecnologías web básicas. Esto ha permitido profundizar en el funcionamiento interno de una aplicación web real, comprendiendo la arquitectura cliente-servidor, la comunicación mediante APIs REST y la gestión de datos persistentes.

FUNCIONALIDAD GENERAL
Soupify permite a los usuarios registrarse e iniciar sesión para acceder a la plataforma. Una vez autenticados, los usuarios pueden explorar el catálogo musical disponible, reproducir canciones mediante un reproductor basado en HTML5 y crear playlists personalizadas en las que organizar su música favorita. Las playlists pueden modificarse en cualquier momento, añadiendo o eliminando canciones, y pueden configurarse como públicas o privadas.

El sistema diferencia entre usuarios estándar y administradores. Los usuarios administradores disponen de funcionalidades adicionales que les permiten gestionar el catálogo musical, añadiendo, editando o eliminando canciones del sistema. Esta diferenciación de roles garantiza un control adecuado del contenido y del funcionamiento general de la aplicación.

TECNOLOGÍAS UTILIZADAS
La aplicación ha sido desarrollada utilizando HTML5 para la estructura del contenido, CSS3 para el diseño visual y JavaScript para la lógica del cliente y la interacción con el usuario. El backend se ha implementado utilizando Node.js o PHP, proporcionando una API REST encargada de gestionar la lógica de negocio y el acceso a la base de datos. Para el almacenamiento persistente de la información se ha utilizado una base de datos SQLite o MySQL. Durante el desarrollo se ha empleado Git para el control de versiones y Visual Studio Code como entorno de desarrollo.

EJECUCIÓN DEL PROYECTO
Para ejecutar Soupify es necesario disponer de un navegador web actualizado y del entorno de ejecución correspondiente al backend utilizado. Una vez descargado el proyecto, se debe configurar la conexión con la base de datos y poner en marcha el servidor backend. Tras ello, la aplicación puede utilizarse accediendo desde el navegador a la dirección local configurada.

USUARIOS DE PRUEBA
Para facilitar la evaluación del proyecto, se han habilitado cuentas de prueba. Existe un usuario con rol administrador y un usuario estándar que permiten comprobar todas las funcionalidades del sistema. Las credenciales de acceso se encuentran indicadas en la documentación del proyecto o pueden configurarse directamente en la base de datos.

ESTADO DEL PROYECTO
Soupify es un proyecto académico finalizado que cumple con los requisitos funcionales y técnicos establecidos en el enunciado de la asignatura. El sistema es completamente funcional en un entorno local y está preparado para futuras ampliaciones y mejoras.

POSIBLES MEJORAS
Como líneas de evolución futura, el proyecto podría ampliarse incorporando estadísticas de uso, recomendaciones automáticas de música, importación y exportación de playlists, un modo oscuro y claro, o mejoras adicionales en las medidas de seguridad para un entorno de producción real.
